# BloomX
BloomX is an interactive motivational device that inspires people in engineering and technology. Shaped like a blooming flower, it combines movement, lighting, and digital feedback to symbolize growth and resilience. Users can track goals and receive positive reinforcement through animations and messages, blending creativity with STEM motivation.
